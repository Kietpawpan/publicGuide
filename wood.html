การตัดไม้ในที่ดินกรรมสิทธิ์

ขั้นตอน

การจัดทำหนังสือสำแดงความถูกต้องด้วยตนเอง

ยินยอมให้กรมป่าไม้เก็บข้อมูลส่วนบุคคล

  • หากยินยอมให้กรมป่าไม้เก็บข้อมูลส่วนบุคคล จะใช้งานระบบจัดทำหนังสือสำแดงความถูกต้องได้
  • หากไม่ยินยอม ท่านสามารถจัดทำหนังสือดังกล่าวด้วยวิธีอื่นตามที่เห็นสมควร โดยให้มีข้อมูลครบถ้วนตามหนังสือกรมป่าไม้ ด่วนที่สุด ที่ ทส 1613.2.3/19527 ลงวันที่ 26 พฤศจิกายน 2564 เรื่อง การรับรองไม้ที่ขึ้นในที่ดินที่มีกรรมสิทธิ์หรือสิทธิครอบครองตามประมวลกฎหมายที่ดิน

เปิดโปรแกรมจัดทำหนังสือสำแดงความถูกต้อง

ระบบ Self-Declaration (SD) เป็น Web Application ของกรมป่าไม้ เข้าใช้งานโดยกดที่ลิงค์ https://etree.forest.go.th/PubblicForm/Attachment3

กรอกข้อมูลใน e-Form

กรอกข้อมูลตามความจริง เพื่อระบุตัวตนของเจ้าของไม้ แหล่งที่มาของต้นไม้ รายการสรุปต้นไม้ที่ตัด บัญชีไม้แนบท้ายหนังสือสำแดงความถูกต้องด้วยตนเอง และแนบไฟล์รูปภาพ

กดปุ่มบันทึกข้อมูล

เพื่อบันทึกข้อมูลลงในเครื่อง Server ของกรมป่าไม้

กดปุ่มพิมพ์

ระบบจะแสดงภาพหนังสือสำแดงความถูกต้องด้วยตนเอง พร้อม QR Code สำหรับตรวจสอบกับต้นฉบับในระบบ SD ของกรมป่าไม้
